# 全国大学生数学竞赛知识库

这是一个面向非数学类全国大学生数学竞赛备考的 Obsidian 知识库，已整合专题总览、精讲笔记、真题索引、公式速查、复习计划和复盘模板。

## 入口

建议用 Obsidian 打开仓库根目录，然后从下面的主页进入：

```text

```

## 使用方式

1. 先读各模块总览，建立题型地图。
2. 再看精讲笔记，掌握高频套路和易错点。
3. 用真题索引做限时训练。
4. 每次训练后写错题复盘，把错因归入具体专题卡片。
5. 考前重点看公式速查、题型反查和错题复盘。
