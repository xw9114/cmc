# 🎯 竞赛解题套路总结

[[00-导航/🏠 数学竞赛主页|← 返回主页]]

> 看到题目→秒判类型→套对方法，这才是竞赛的核心能力。

---

## 一、填空题做题流程

```
读题 → 判断"大类" → 代入特殊值验算 → 写答案
          ↓
    极限 / 导数 / 积分 / 级数 / 矩阵
```

**常见陷阱**：
- 极限：洛必达适用条件（需验证 $f'/g'$ 极限存在）
- 积分：对称性判断前先确认区间是否关于原点对称
- 级数：收敛半径端点单独判断
- 矩阵：特征值与行列式的关系

## 二、看到这个词/结构，用这个方法

| 看到... | 想到... |
|---------|---------|
| $\lim \frac{\sin f(x)}{g(x)}$，$f,g\to 0$ | 等价无穷小 $\sin f\sim f$ |
| $\lim (1+\alpha)^\beta$，$\alpha\to 0$ | $e^{\lim\alpha\beta}$ |
| $\lim \frac{\text{含}\ln/e^\cdot}{\text{多项式}}$ | Taylor展开到需要阶 |
| $\frac{1}{n}\sum f(k/n)$ | Riemann和 |
| 递推数列 | 单调有界原理 |
| $\int_{-a}^a f(x)$ | 奇偶性（直接为0或加倍）|
| $\int_0^\pi xf(\sin x)$ | 令 $x=\pi-t$，区间再现 |
| 曲线积分 $\oint P\,dx+Q\,dy$ | Green公式（先验 $Q_x=P_y$?）|
| 包含 $x^2+y^2+z^2$ 的三重积分 | 球坐标 |
| 行列式含参数 | 展开看特征多项式或换行列 |
| 方程有根 | 零点定理 / Rolle |
| 证明不等式 | 构造 $f(x)$ 令 $f(0)=0$，证 $f'\geq 0$ |
| 证 $f'(\xi)=g(\xi)$ | 构造 $F(x)=f-G$，用Rolle |

## 三、填空题快速判断答案合理性

| 方法 | 操作 |
|------|------|
| 量纲检验 | 极限值量纲应与函数量纲一致 |
| 代特殊值 | 令 $x=0$ 或 $n=1$ 验证 |
| 正负号 | 被积函数正→积分正；极限趋0→从正/负侧趋 |
| 数量级 | 结果是否在合理范围（比如积分$\leq$最大值×区间长）|

## 四、常见错误防范

### ❌ 错误1：等价替换用于加减

**错**：$\sin x - \tan x\sim x-x=0$（不能这样！）

**正确**：$\sin x-\tan x=\sin x\left(1-\frac{1}{\cos x}\right)\sim x\cdot\frac{-1+\cos x}{\cos x}\sim x\cdot(-\frac{x^2}{2})=-\frac{x^3}{2}$

---

### ❌ 错误2：洛必达适用条件忽略

$\lim_{x\to\infty}\frac{x+\sin x}{x}$：若用L'Hôpital → $\lim\frac{1+\cos x}{1}$ 不存在！

但原极限 $=\lim(1+\frac{\sin x}{x})=1$（正确做法：直接分拆）

---

### ❌ 错误3：Green公式忽略奇点

$\oint_L \frac{-y\,dx+x\,dy}{x^2+y^2}$ 在含原点的区域上不能直接用Green公式！

---

### ❌ 错误4：幂级数收敛域端点遗漏

比值法给出 $|x|<R$，但 $x=\pm R$ 时需单独代入判断。

---

### ❌ 错误5：递推数列只设极限不证收敛

必须先证明收敛，再取极限解方程。

---

## 五、竞赛评分常见扣分点

1. **中值定理**未说明 $f$ 满足连续/可导条件
2. **Taylor公式**未指明余项阶数
3. **Green公式**未说明区域单连通 / 曲线方向
4. **级数**判别法未写出 $\rho<1$ 故收敛的结论
5. **积分换元**未换积分限（定积分）

## 六、时间分配建议（120分钟）

| 题型 | 建议时间 | 说明 |
|------|---------|------|
| 5道填空（30分） | 25分钟 | 每题5分钟，不确定先跳过 |
| 4道计算（60分） | 60分钟 | 每题15分钟 |
| 2道证明（30分） | 30分钟 | 每题15分钟 |
| 检查 | 5分钟 | 重点检查填空答案 |

**策略**：
- 填空不会的先跳过，大题反而容易拿分
- 证明题写出思路和关键步骤，即使没证完也有部分分
- 计算题写完整过程，别只写结果

---

## 七、草稿纸布局建议

```text
左上：填空题快速计算
右上：公式与临时结论
下半：大题完整演算
边角：待回看题号
```

- 同一题不要把关键步骤分散在多页；
- 参数、积分上下限、正负号单独圈出；
- 最终答案写回试卷前做一次量级检查。

## 八、证明题保底得分模板

即使暂时证不完整，也应写出：

1. 所使用定理及条件；
2. 辅助函数或核心变形；
3. 关键中间结论；
4. 最终目标与中间结论之间的联系。

## 九、错题复盘四问

- **错在哪里**：概念、方法、计算还是时间分配？
- **为什么错**：哪个条件或结构没有识别？
- **以后看到什么信号要警觉**？
- **能否把本题压缩成一句套路**？
