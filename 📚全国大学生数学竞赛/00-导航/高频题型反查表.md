# 🔍 高频题型反查表

[[00-导航/🏠 数学竞赛主页|← 返回主页]]

> 从题目外形反查方法，而不是从章节顺序死记。

| 题目结构/关键词 | 第一反应 | 检查点 |
|---|---|---|
| 根式相减的极限 | 有理化 | 共轭后是否降阶 |
| 加减项大量抵消 | Taylor | 展开到首个不抵消项 |
| 前 $n$ 项和除以 $n^p$ | Stolz | 分母单调趋无穷 |
| $\frac1n\sum f(k/n)$ | Riemann 和 | 区间与步长 |
| 递推数列 | 单调有界 | 先证收敛 |
| $f'(\xi)=\lambda f(\xi)$ | 指数因子 | Rolle 条件 |
| 方程至少有根 | 零点定理 | 连续与异号 |
| 方程根唯一 | 单调性 | 导数严格同号 |
| $\int_{-a}^{a}$ | 奇偶性 | 区间对称 |
| $\int_0^a xf(a-x)$ | 区间再现 | 换元后与原式相加 |
| 根式积分 | 三角换元 | 换限或回代 |
| 反常积分 | 比较/Dirichlet | 瑕点与无穷端 |
| 圆域二重积分 | 极坐标 | Jacobian $r$ |
| 封闭曲线积分 | Green | 方向与奇点 |
| $P_y=Q_x$ | 路径无关 | 区域单连通 |
| 正项级数含阶乘 | 比值法 | $\rho=1$ 时换方法 |
| 交错级数 | Leibniz | 再判绝对收敛 |
| 幂级数 | 半径+端点 | 两端单独代入 |
| 二阶常系数 ODE | 特征方程 | 重根/复根 |
| 矩阵含参数 | 秩或行列式 | 分类讨论参数 |
| 实对称矩阵 | 正交对角化 | 特征向量正交化 |
