## 技巧一：对称区间奇偶性（秒杀题）

$$\int_{-a}^a f(x)\,dx=\begin{cases}0 & f\text{ 为奇函数}\\2\int_0^a f(x)\,dx & f\text{ 为偶函数}\end{cases}$$

### 例1 ⭐（极速）

$$\int_{-\pi}^{\pi}\frac{x\sin x}{1+\cos^2 x}\,dx$$

**解**：$f(x)=\dfrac{x\sin x}{1+\cos^2 x}$，$f(-x)=\dfrac{-x\cdot(-\sin x)}{1+\cos^2 x}=f(x)$，偶函数

$$=2\int_0^\pi\frac{x\sin x}{1+\cos^2 x}\,dx$$

令 $x=\pi-t$：$\int_0^\pi\frac{x\sin x}{1+\cos^2 x}dx=\int_0^\pi\frac{(\pi-t)\sin t}{1+\cos^2 t}dt$

两式相加：$2\int_0^\pi\frac{x\sin x}{1+\cos^2 x}dx=\pi\int_0^\pi\frac{\sin t}{1+\cos^2 t}dt=\pi\Big[-\arctan(\cos t)\Big]_0^\pi=\pi\cdot\frac{\pi}{2}$

原式 $=2\cdot\frac{\pi^2}{4}=\boxed{\dfrac{\pi^2}{2}}$

## 技巧二：区间再现（$x\to a-x$ 换元）

**方法**：若被积函数满足 $f(a-x)+f(x)=c$（常数），则

$$\int_0^a f(x)\,dx=\frac{c\cdot a}{2}$$

### 例2 ⭐⭐

$$\int_0^\pi\frac{x\sin x}{1+\sin^2 x}\,dx$$

**解**：令 $x=\pi-t$，$I=\int_0^\pi\frac{(\pi-t)\sin t}{1+\sin^2 t}dt=\pi\int_0^\pi\frac{\sin t}{1+\sin^2 t}dt-I$

$$2I=\pi\int_0^\pi\frac{\sin x}{1+\sin^2 x}dx$$

令 $u=\cos x$，则

$$\int_0^\pi\frac{\sin x}{1+\sin^2x}\,dx
=\int_{-1}^{1}\frac{du}{2-u^2}
=\sqrt2\ln(\sqrt2+1).$$

因此

$$I=\boxed{\dfrac{\pi}{\sqrt2}\ln(\sqrt2+1)}.$$

## 技巧三：分部积分与表格法

**口诀**：反对幂指三（反三角、对数、幂函数、指数、三角），从左到右优先作 $u$

### 例3 ⭐⭐

$$\int x^2 e^x\,dx$$

**表格法**：

| 求导 | 积分 |
|------|------|
| $x^2$ | $e^x$ |
| $2x$ | $e^x$ |
| $2$ | $e^x$ |
| $0$ | $e^x$ |

$$=(+)x^2 e^x+(-)2xe^x+(+)2e^x+C=e^x(x^2-2x+2)+C$$

---

### 例4 ⭐⭐⭐（竞赛常见）

$$I_n=\int_0^{\pi/2}\sin^n x\,dx \quad\text{（华里士/Wallis公式）}$$

**递推推导**：$I_n=\dfrac{n-1}{n}I_{n-2}$（分部积分）

$$I_n=\begin{cases}\dfrac{(n-1)!!}{n!!}\cdot\dfrac{\pi}{2} & n\text{ 为偶数}\\\dfrac{(n-1)!!}{n!!} & n\text{ 为奇数}\end{cases}$$

**常用值**：
- $I_0=\pi/2$，$I_1=1$，$I_2=\pi/4$，$I_3=2/3$，$I_4=3\pi/16$

## 技巧四：换元法选择

### 例5 ⭐⭐（三角换元）

$$\int\frac{dx}{\sqrt{x^2-a^2}}\quad(x>a>0)$$

令 $x=a\sec\theta$：$\sqrt{x^2-a^2}=a\tan\theta$，$dx=a\sec\theta\tan\theta\,d\theta$

$$=\int\frac{a\sec\theta\tan\theta\,d\theta}{a\tan\theta}=\int\sec\theta\,d\theta=\ln|\sec\theta+\tan\theta|+C=\ln\left|x+\sqrt{x^2-a^2}\right|+C$$

---

### 例6 ⭐⭐⭐（含参变量积分）

$$F(t)=\int_0^t e^{-x^2}\,dx,\quad\text{求}F'(t)$$

$F'(t)=e^{-t^2}$（变限积分求导）

**进阶**：$G(t)=\int_0^{t^2}e^{-x^2}\,dx$，$G'(t)=e^{-t^4}\cdot 2t$（链式法则）

**再进阶**：$H(t)=\int_t^{t^2}e^{-x^2}\,dx$，$H'(t)=e^{-t^4}\cdot 2t-e^{-t^2}$

## 技巧五：反常积分敛散性

### 例7 ⭐⭐（比较法）

判断 $\displaystyle\int_1^{+\infty}\frac{\sin x}{x^p}\,dx$（$p>0$）的敛散性

- $p>1$：绝对收敛（$|\sin x/x^p|\leq 1/x^p$，$p>1$ 收敛）
- $0<p\leq 1$：条件收敛（Dirichlet判别法，分部积分：$\frac{\sin x}{x^p}$ 收敛；但 $\int\frac{|\sin x|}{x^p}dx$ 发散）

**Dirichlet判别法**：$\int_a^{+\infty}f(x)g(x)\,dx$ 收敛，若
- $F(x)=\int_a^x f(t)\,dt$ 有界
- $g(x)$ 单调趋0

## 技巧六：积分估计（竞赛常用）

### 例8 ⭐⭐

证明：

$$\frac{\pi}{4}<\int_0^1\frac{dx}{\sqrt{1+x^4}}<1.$$

**证明**：

当 $0<x<1$ 时，

$$(1+x^2)^2=1+2x^2+x^4>1+x^4,$$

故

$$\frac{1}{\sqrt{1+x^4}}>\frac{1}{1+x^2}.$$

两边在 $[0,1]$ 上积分：

$$\int_0^1\frac{dx}{\sqrt{1+x^4}}
>\int_0^1\frac{dx}{1+x^2}
=\frac{\pi}{4}.$$

另一方面，$1+x^4>1$，所以

$$\frac{1}{\sqrt{1+x^4}}<1,$$

从而

$$\int_0^1\frac{dx}{\sqrt{1+x^4}}<1.$$

$\blacksquare$

> [!tip] 积分估计优先寻找“积分后正好得到目标常数”的比较函数。本题目标含 $\pi/4$，自然想到 $\int_0^1\frac{dx}{1+x^2}$。

## 📋 积分速查

| 类型 | 换元/方法 |
|------|---------|
| $\sqrt{a^2-x^2}$ | $x=a\sin\theta$ |
| $\sqrt{a^2+x^2}$ | $x=a\tan\theta$ |
| $\sqrt{x^2-a^2}$ | $x=a\sec\theta$ |
| $R(\sin x,\cos x)$ 奇次 | 令分子 $t=\cos x$ 或 $\sin x$ |
| $R(\sin x,\cos x)$ 一般 | 万能 $t=\tan(x/2)$ |
| 幂函数×指数函数 | 分部（幂函数求导） |
| 幂函数×对数函数 | 分部（对数函数求导） |
| 对称区间 | 奇偶性判断 |

## 🧭 定积分方法决策树

```text
先看区间
├─ [-a,a] → 检查奇偶性
├─ [0,a] 且含 x 与 a-x → 区间再现
├─ 被积函数含参数 → 积分号下求导 / 变限积分
├─ 根式结构 → 三角换元
├─ 多项式×指数/三角/对数 → 分部积分
└─ 反常积分 → 先定位瑕点，再判敛散
```

## ❌ 高频失分点

- 定积分换元后忘记改变上下限；
- 分部积分漏掉边界项；
- 反常积分未写成极限；
- 使用奇偶性前未确认积分区间关于原点对称；
- 只比较函数值，不确认比较方向。
