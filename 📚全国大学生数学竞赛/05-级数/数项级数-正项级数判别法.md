# 数项级数-正项级数判别法

[[05-级数/级数总览|← 返回总览]]

> 正项级数判别的核心是匹配增长速度。阶乘和指数优先比值法，幂函数优先比较法或 $p$ 级数。

---

## 常用判别法

| 判别法 | 适用结构 | 结论 |
|---|---|---|
| 比较判别法 | 与 $1/n^p$、几何级数同阶 | 大收敛推小收敛，小发散推大发散 |
| 极限比较法 | 能看出主阶 | $\lim a_n/b_n=c>0$ 同敛散 |
| 比值法 | 含 $n!$、指数、连乘 | $\rho<1$ 收敛，$\rho>1$ 发散 |
| 根值法 | 含 $n$ 次幂 | $\rho<1$ 收敛，$\rho>1$ 发散 |
| 积分判别法 | $a_n=f(n)$ 且单调正 | 与反常积分同敛散 |

---

## 方法流程

1. 先确认 $a_n\ge0$。
2. 看结构：阶乘/指数/幂函数/根式/对数。
3. 先做必要条件：若 $a_n\nrightarrow0$，级数发散。
4. 选择判别法。
5. 比值法或根值法遇到 $\rho=1$ 时，必须换方法。

---

## 典型例题

判断

$$\sum_{n=1}^{\infty}\frac{n!}{n^n}$$

的敛散性。

设

$$a_n=\frac{n!}{n^n}.$$

用比值法：

$$\frac{a_{n+1}}{a_n}
=\frac{(n+1)!}{(n+1)^{n+1}}\cdot\frac{n^n}{n!}
=\left(\frac{n}{n+1}\right)^n.$$

因此

$$\lim_{n\to\infty}\frac{a_{n+1}}{a_n}=e^{-1}<1.$$

所以级数收敛。

---

## 判别优先级

```text
先看 a_n 是否趋于 0
→ 阶乘/指数：比值法
→ n 次幂：根值法
→ 有理式/根式：极限比较
→ 对数慢变量：比较或积分判别
→ 比值/根值等于 1：换方法
```

---

## 易错清单

- 忘记先验 $a_n\to0$；
- 比值法 $\rho=1$ 时误判；
- 比较判别方向写反；
- 正项级数方法误用于变号级数；
- 对数因子影响临界情形，例如 $\sum 1/(n\ln n)$。
