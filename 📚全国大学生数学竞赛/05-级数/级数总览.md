## 常用级数求和

$$\sum_{n=1}^\infty \frac{1}{n^2}=\frac{\pi^2}{6}, \quad \sum_{n=0}^\infty x^n = \frac{1}{1-x}\ (|x|<1)$$

$$e^x=\sum_{n=0}^\infty\frac{x^n}{n!},\quad \ln(1+x)=\sum_{n=1}^\infty\frac{(-1)^{n-1}}{n}x^n$$

$$\sin x=\sum_{n=0}^\infty\frac{(-1)^n}{(2n+1)!}x^{2n+1},\quad \cos x=\sum_{n=0}^\infty\frac{(-1)^n}{(2n)!}x^{2n}$$

## 🧭 敛散性判断顺序

```text
先检查 a_n → 0？
├─ 否 → 发散
└─ 是
   ├─ 正项 → 比值/根值/比较/积分
   ├─ 交错 → Leibniz，再判绝对收敛
   └─ 振荡 → Dirichlet/Abel
```

## 📌 幂级数三步

1. 求收敛半径 $R$；
2. 得到开区间 $|x-x_0|<R$；
3. 将两个端点分别代入原级数判断。

> [!warning] 收敛半径不等于收敛区间，端点永远需要单独判断。
