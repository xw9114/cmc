## 题型分类

### 一阶方程
- [[ODE-可分离变量方程]] — $f(y)dy = g(x)dx$
- [[ODE-齐次方程]] — 令 $y=vx$，化为可分离
- [[ODE-一阶线性方程]] — 公式法，积分因子
- [[ODE-Bernoulli方程]] — $y'+P(x)y=Q(x)y^n$，令 $z=y^{1-n}$
- [[ODE-全微分方程]] — 判断 $\frac{\partial P}{\partial y}=\frac{\partial Q}{\partial x}$

### 二阶及高阶方程
- [[ODE-二阶常系数齐次]] — 特征方程法（三种根的情况）
- [[ODE-二阶常系数非齐次]] — 特解形式（多项式型、指数型、三角型）
- [[ODE-Euler方程]] — 令 $x=e^t$ 化为常系数
- [[ODE-降阶法]] — 缺 $y$ 型（令 $p=y'$）、缺 $x$ 型

### 方程组与应用
- [[ODE-线性方程组]] — 消元法、矩阵指数法
- [[ODE-初值问题]] — 代入初始条件定常数
- [[ODE-建模问题]] — 几何意义、物理背景

## 高频考点

| 题型 | 关键步骤 |
|------|---------|
| 一阶线性 | 公式 $y=e^{-\int P}\left(\int Qe^{\int P}dx+C\right)$ |
| 二阶常系数齐次 | 特征根 → 写通解 |
| 非齐次特解 | 判断右端类型 → 设特解形式 |
| 几何/物理建模 | 正确建立微分方程是关键 |

## 🧭 一阶方程识别

| 形式 | 方法 |
|---|---|
| $y'=f(x)g(y)$ | 分离变量 |
| $y'=F(y/x)$ | 令 $y=vx$ |
| $y'+P(x)y=Q(x)$ | 积分因子 |
| $y'+Py=Qy^n$ | Bernoulli，令 $z=y^{1-n}$ |
| $Mdx+Ndy=0,\ M_y=N_x$ | 全微分 |

## ❌ 易错清单

- 分离变量时除以含 $y$ 的式子，可能丢失常数解；
- 求通解后忘记代初值；
- 非齐次特解与齐次解重复时要乘 $x^s$；
- 特征根为 $\alpha\pm i\beta$ 时，通解含 $\cos\beta x,\sin\beta x$。
